## carnival-spring-boot-starter-springutils

本插件包含若干个工具类供使用，当应用中无法使用依赖注入时，本插件可以派上用场。

* [SpringUtils](https://github.com/yingzhuo/carnival/blob/master/carnival-spring-boot-starter-springutils/src/main/java/com/github/yingzhuo/carnival/spring/SpringUtils.java)
* [ProfileUtils](https://github.com/yingzhuo/carnival/blob/master/carnival-spring-boot-starter-springutils/src/main/java/com/github/yingzhuo/carnival/spring/ProfileUtils.java)
* [ArgumentUtils](https://github.com/yingzhuo/carnival/blob/master/carnival-spring-boot-starter-springutils/src/main/java/com/github/yingzhuo/carnival/spring/ArgumentUtils.java)

本插件在classpath中自动生效，无需配置。
