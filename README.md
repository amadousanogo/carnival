下载
---

本项目没有发布到`maven`中央仓库，需自行安装到本地库。

```bash
#!/usr/bin/env bash

git clone https://github.com/yingzhuo/carnival.git
mvn -f ./carnival/pom.xml clean install -Dmaven.test.skip=true
```
